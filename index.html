SQL — Database Querying & Extraction
SQL is the fundamental language of data analysis. Before any chart gets built, the right data must be extracted cleanly. I write queries that filter, join, aggregate, and summarise data from relational databases — delivering exactly the dataset you need for any report, without bloated exports or manual cleanup afterwards.
  • Core querying: SELECT, WHERE, GROUP BY, ORDER BY, HAVING
  • INNER, LEFT, and multi-table JOINs to merge data across sources
  • Aggregate functions: COUNT, SUM, AVG, MIN, MAX with GROUP BY
  • Subqueries and Common Table Expressions (CTEs) for multi-step logic
  • Window functions: ROW_NUMBER, RANK, running totals, rolling averages
